# Verilog-Vision
Vision system written in verilog, run on Kria board

Repo includes all files necessary to run project. It includes source files for all functions of vision system which are in /vision_sources folder. Testbench is located in tb_sources - currently it only works for 64 by 64 pixel images. Ips include source files that were added as user-defined ip to original vivado project. You can either make new ip with them or import them to project directly.
